# Linear Regression – Insurance Dataset

## Project Overview
This project implements **Linear Regression** to predict medical insurance charges using a simple insurance dataset.

The goal of this project is to understand the basic machine learning workflow using supervised learning.

---

## Dataset
The dataset contains information about insurance customers such as:
- age
- sex
- bmi
- children
- smoker
- region

**Target variable:**
- charges

---

## Steps Performed
1. Loaded the insurance dataset using Pandas
2. Converted categorical columns into numerical values
3. Selected features (X) and target (y)
4. Split data into training and testing sets
5. Trained a Linear Regression model
6. Evaluated the model using MAE, RMSE, and R² score

---

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn

---

## Output
The model predicts insurance charges and is evaluated using regression metrics.
